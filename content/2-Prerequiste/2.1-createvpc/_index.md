#### Create VPC
1. Access **VPC.**

- Select **your VPC**
- Click **Create VPC**

{{< img src="/images/2.prerequisite/Chuanbi-(1).png" alt="VPC" >}}

2. Options in **VPC Wizard**

- Select **VPC and more**
- Enter **VPC** name
- Number of **AZ**: 1
- Number of **private/public** subnets: 1

{{< img src="/images/2.prerequisite/Chuanbi-(2).png" alt="VPC" >}}

3. Click **Create VPC**
