#### Create VPC
1. Access **VPC.**

- Select **your VPC**
- Click **Create VPC**

![VPC](/images/2.prerequisite/Chuanbi-(1).png)

2. Options in **VPC Wizard**

- Select **VPC and more**
- Enter **VPC** name
- Number of **AZ**: 1
- Number of **private/public** subnets: 1

![VPC](/images/2.prerequisite/Chuanbi-(2).png)

3. Click **Create VPC**
